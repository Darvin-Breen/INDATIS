# R Scripts for ESG Analysis

## Files
- `TFIDF_IT.R` - TF-IDF clustering analysis for Italian sustainability reports

## Description
This script performs the original TF-IDF analysis on the sustainability reports, creating document-term matrices and clustering using k-means.
